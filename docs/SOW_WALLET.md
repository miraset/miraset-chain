# SOW_WALLET 
### statement of work for MIRASET wallet development.

Задача: нужно сделать кошелек для MIRASET, который будет работать на блокчейне Sui. Этот кошелек должен поддерживать все функции, необходимые для управления MIRASET токенами, включая отправку, получение и хранение токенов.
В данном случае, мы будем использовать Sui SDK для разработки кошелька, при условии что есть возможность задать провайдер кастомный.

Установщик: 

Программа должна распаковаться условно в папку C:\MIRASET_WALLET

структура папки должна быть следующая:
miraset.exe // client
wallet-miraset.exe // gui кошелек


При запуске wallet-miraset.exe, пользователь должен увидеть интерфейс кошелька, который позволяет ему управлять своими MIRASET токенами. Интерфейс должен быть интуитивно понятным и простым в использовании.

Функциональность кошелька должна включать в себя следующие возможности:
1. Просмотр баланса MIRASET токенов.
2. Отправка MIRASET токенов другим пользователям.
3. Получение MIRASET токенов от других пользователей.
4. Хранение MIRASET токенов в безопасном месте.
5. Поддержка нескольких аккаунтов для управления различными адресами.

Обращение на API/RPC miraset клиента.

Wallet исходник в ./wallet
